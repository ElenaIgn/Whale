## 📌 Описание проекта
Данный репозиторий содержит минимально жизнеспособный продукт (**MVP**) программно-аппаратного комплекса для автоматического обнаружения и мониторинга особей гренландского кита (*Balaena mysticetus*) по материалам аэрофотосъемки с БПЛА. 

Проект разработан в рамках выпускной квалификационной работы (**ВКР**) магистратуры **Томского государственного университета (ТГУ)**. 

### 🌟 Ключевые особенности системы:
* 🎯 **Высокоточная локализация:** Использование легковесной нейросети **YOLOv8n** для детекции объектов на водной поверхности.
* 🛠️ **Уникальный датасет ТГУ:** Набор данных из **459 кадров**, размеченный с помощью инструментов **CVAT** и **X-AnyLabeling**.
* ⚡ **Асинхронный GUI:** Графический интерфейс оператора на **CustomTkinter** с поддержкой Dark Mode, работающий без зависаний благодаря многопоточности (`threading`).
* 🔋 **Оптимизация под Edge AI:** Архитектура адаптирована для развертывания на микрокомпьютере **NVIDIA Jetson Orin NX** с энергопотреблением **<15 Вт**.

## 📊 Структура уникального датасета
В ходе исследования сырые видеоматериалы были очищены от дубликатов и распределены автоматическим скриптом случайного разбиения (`seed=42`) в пропорции **75% / 25%**:

* 📁 **Train (Обучающая выборка):** 344 кадра с файлами разметки YOLO (`.txt`).
* 📁 **Val (Валидационная выборка):** 115 кадров с файлами разметки YOLO (`.txt`).
* 🏷️ **Целевой класс:** `0: whale` (конфигурация зафиксирована в `dataset_config.yaml`).

## 📈 Количественные результаты тестирования


| Метрика | Значение на CPU (Тестовый стенд) | Значение на Jetson Orin NX (FP16)* |
| :--- | :---: | :---: |
| 🎯 **Precision** (Точность) | **0.91** | 0.90 |
| 📊 **Recall** (Полнота) | **0.87** | 0.86 |
| 💎 **F1-Score** | **0.89** | 0.88 |
| ⏱️ **Latency** (Задержка) | 420 мс | **280 мс** |
| ⚡ **Энергопотребление** | — | **12.5 Вт** (Лимит <15 Вт) |

⚠️ **Известные ограничения MVP:** Из-за небольшого числа эпох обучения (30 эпох на CPU) модель может пропускать сильно погруженных под воду особей или давать ложные срабатывания на глубоких тенях от крупных волн при сильном шторме. Система работает в светлое время суток и требует ИК/тепловизионного оборудования для работы в тумане или полярную ночь.

## 🛠️ Инструкция по установке и запуску

### 1. Подготовка окружения


```bash
# Клонирование репозитория
git clone https://github.com
cd whale-identification-system

# Создание и активация виртуального окружения
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
venv\Scripts\activate     # Для Windows

# Установка пакетов
pip install ultralytics customtkinter opencv-python pillow
```

### 2. Запуск графического интерфейса
Для старта операторского приложения введите команду:
```bash
python main_gui.py
```
📐 **Как пользоваться:** 
1. Нажмите кнопку **«Загрузить видео»** и выберите ваш видеофайл (`.mp4` / `.avi`).
2. Нажмите кнопку **«Распознавание кита»** для запуска нейросетевого инференса.
3. Если модель пропускает китов, сместите слайдер **«Порог уверенности»** влево (до уровня `0.05 - 0.10`).
